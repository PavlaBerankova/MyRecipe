# MyRecipe app

## Application with my recipes by category or show you all recipes.

I learned in this project:

 - JSON parse
 - MVVM architecture
 - @ObservedObject, @ObserableObject, @Published
 - NavigationStack, NavigationLink
 - TabView
 - Grid Layout
